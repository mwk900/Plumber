# Cresco Trades Nottingham Demo Site

Production-style local trades landing site built with Next.js (static export), TypeScript, Tailwind CSS, and Framer Motion.

## Run locally

```bash
npm install
npm run dev
```

## Build static export

```bash
npm run build
```

The static output is generated for easy deployment to Netlify or Cloudflare Pages.

## Editable content

Update business details, services, areas, trust items, and reviews in:

- `src/data/site.json`

> Replace placeholder phone, email, address, certifications, and sample reviews before publishing.
